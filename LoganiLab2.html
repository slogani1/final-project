Terms and Definitions

How do you define a table in html?
Using the <table> element.
<tr>, <th>, <td>
<tr> defines a row in a table, <th> defines a heading cell in a table, and <td> defines only a cell in a table.
<thead>, <tfoot>, <caption>
<thead> groups heading content within a table together, <tfoot> groups all the footer content within a table togehter, and <caption> defines or labels a table with a specific caption.
Colspan vs. Rowspan
Colspan takes two horizontally adjacent table cells and combines them into one cell, while Rowspan does the same for two vertically adjacent cells.
Cell spacing vs Padding
While cell spacing defines the space within any given table cell, padding only defines the space between any given cell's boarder and any content present within the cell.
Table Borders in CSS
To define a table's border in CSS, you must first specify the class or element whose border needs to be changed, then use the border property to assign the border values like pixel border or solid vs. dashed. Additionally, the border-width property can be used to define the border width specifically.
